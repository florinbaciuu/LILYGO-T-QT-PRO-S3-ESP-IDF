# SYS Command

[![Component Registry](https://components.espressif.com/components/esp-qa/sys-cmd/badge.svg)](https://components.espressif.com/components/esp-qa/sys-cmd)

This repository contains some basic system related commands based esp-idf console.

There is no GitHub repository for this component. Please report issues to [the iperf-cmd repository](https://github.com/espressif/iperf-cmd) instead.

### Installation

- To add a plugin command or any component from IDF component manager into your project, simply include an entry within the `idf_component.yml` file.

  ```yaml
  dependencies:
    esp-qa/sys-cmd:
      version: "^0.1.4"
  ```
- For more details refer [IDF Component Manager](https://docs.espressif.com/projects/idf-component-manager/en/latest/)
